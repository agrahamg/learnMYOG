Getting Started

  • One zipper atleast Fabric Height . For more information on zippers, see Zipper Theory.
  • One fabric panel to match the pattern dimensions.

There are tons of fabric options. Here are a few recommendations.

  • Cordura 500d or 210d HDPE Gridstop are great choices for durable, self standing zip pouch.
  • 1.9oz PU Coated Nylon for lightweight, water resistant pouches with many color options.
  • Patterned cotton have unlimited creative options and can be found at local quilt shops.

Assembly Instructions

Subheading

  1. Using the generated Fabric Width and Fabric Height , measure and draw the dimensions on a fabric panel.
  2. Cut the fabric panel along the outer perimeter.
  3. Mark the 1/2 inch zipper allowance along the Fabric Height edges.
  4. Mark the 3/8 inch seam allowance along the Fabric Width edges.
